# Thư mục lưu trữ biểu mẫu bảng biểu (LaTeX Tables Assets)

Thư mục này chứa danh sách các bảng biểu dữ liệu mẫu và các kết quả phân tích số liệu thực nghiệm.

## Các bảng biểu chính đã được định nghĩa trong báo cáo LaTeX:

1. **Bảng 1.1 (Chương 1)**: Phân công nhiệm vụ chi tiết và mức độ hoàn thành công việc của từng thành viên trong nhóm (Đỗ Đức Huy, Trương Việt Anh, Đặng Thái Sơn) đối khớp theo nhật ký kỹ thuật.
2. **Bảng 2.1 (Chương 2)**: So sánh đặc tính kỹ thuật của nguyên mẫu nhóm thiết kế với các giải pháp hiện có trên thị trường (PAMAir, IQAir, DIY thông thường).
3. **Bảng 2.2 (Chương 2)**: So sánh chi tiết ưu/nhược điểm giữa hai giao thức HTTPS POST (REST API Supabase) và MQTT truyền thống trong phạm vi đề tài.
4. **Bảng 2.3 (Chương 2)**: Bảng đặc tả yêu cầu chức năng của trạm đo (FR1 - FR7).
5. **Bảng 2.4 (Chương 2)**: Bảng đặc tả yêu cầu phi chức năng về hiệu năng, an toàn đa luồng, bảo mật token và chi phí (NFR1 - NFR12).
6. **Bảng 4.1 (Chương 4)**: Bảng phân bổ chân GPIO thực tế kết nối trên chip nhúng ESP32 đối khớp theo mã nguồn `Config.h`.
7. **Bảng 4.2 (Chương 4)**: Bảng kết quả chạy các kịch bản kiểm thử (Test cases) chức năng hệ thống cục bộ, ngắt mạng và khôi phục hoạt động.
8. **Bảng 4.3 (Chương 4)**: Bảng so sánh sai số đánh giá (MAE, RMSE, $R^2$) của dữ liệu bụi thô và các thuật toán học máy hiệu chuẩn (Linear Regression, Random Forest).
9. **Bảng Phụ lục C**: Mẫu trích xuất chuỗi dữ liệu thực nghiệm đo liên tục trong phòng học và khu vực nhà xe nhà xe ngoài trời.
